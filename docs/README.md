## Brand assets

All project graphics live in `docs/assets/reach/`:

- `docs/assets/reach/logos/` - primary logo and wordmark
- `docs/assets/reach/icons/` - UI icons (server, client, tunnel, webhook, security)
- `docs/assets/reach/banners/` - hero/banner art
- `docs/assets/reach/ui/` - starter UI theme tokens (`theme.css`)

Use the SVGs directly in the docs and web UI for crisp scaling.
